KAKURO SOLVER

Версия 1.4
Автор: Овсянникова Дарина

Описание
Решатель какуро. 

Доп. задачи:
1) Добавлен графический интерфейс с возможностью играть.
Доступны три уровня сложности: Easy, Medium, Hard.
Доступна функция проверки правильности решения.
2) Добавлена кнопка "Показать правильное решение".
Добавлены клиент и сервер. Сервер по запросу клиента отсылает ему 
один из какуро, хранящихся у него.
3) Добавлена генератор какуро. Теперь при выборе уровня
сложности не загружает карту, а генерирует ее в соответствии с выбранным уровнем сложности.
